💰 재원 요약 (어디서 얼마를 회수하나)

회수가능액(옮길 돈)은 실지출이라 확실하고, 기회손실(임팩트)은 '평소 효율이었다면' 가정이 섞인 추정치.

📑 재배분안 (우선순위 순)
1순위메타광고 → 네이버광고 · 광고 증액(임팩트 8,822,424원 × 빈도 1주)
1. 문제 정의 메타광고 과집행: 회수 가능한 저효율 지출.
2. 근거 데이터 회수가능액 3,194,281원 / 재원채널 평소 ROAS 2.59 / 수혜: ROAS 6.59 — 시장 벤치마크 상위25% 이상(증액 여력 검증)
3. 재배분안 메타광고 → 네이버광고, 3,194,281원(테스트 상한), 점진 증액 후 재측정
4. 기대효과·한계 순증 매출 상한 약 12,772,685원 (선형확장 가정). 한계수익 체감으로 실제는 하회 가능 — 소액 테스트로 검증 후 확대.
2순위메타광고 → 오가닉 · 전략 투자(SEO·콘텐츠)(임팩트 8,822,424원 × 빈도 1주)
1. 문제 정의 메타광고 과집행: 회수 가능한 저효율 지출.
2. 근거 데이터 회수가능액 3,194,281원 / 재원채널 평소 ROAS 2.59 / 수혜: 광고비 직접지출 0(무료 아님 — 과거 유료광고 낙수효과·SEO 누적 결과), 매출 104,238,814(전체 23.71%), CVR 3.11%
3. 재배분안 메타광고 → 오가닉, 3,194,281원 범위 내 SEO·콘텐츠 투자(광고비 직접 배정 아님)
4. 기대효과·한계 오가닉은 ROAS 측정불가라 원 단위 추정 없이 정성 기대(중장기 유입 성장). 효과 발현에 시차 존재.
재원 공유 주의 — 위 안들은 같은 재원(과집행 회수분)을 공유한다. 1순위를 우선 집행하고 성과 확인 후 다음 순위로 순차 배분한다.
⚖️ 우선순위 판단 기준 (설계 근거)
  1. 1차 = 임팩트 × 빈도. 임팩트만 보면 1주 대형사고와 매주 새는 구멍을 못 가린다. 빈도를 곱해 반복되는 구조적 낭비를 위로.
  2. 2차(동점) = 수혜처 즉효성. 같은 재원 공유 시 점수가 같아, 즉시 ROAS가 실현되는 광고 증액을 전략 투자보다 우선.
  3. 옮기는 금액 = 회수가능액(≠임팩트). 실제 회수 가능한 초과지출만 옮긴다. 임팩트로 옮기면 허수 기획.
  4. 수혜처 = 절대 ROI가 아니라 시장 벤치마크 정성 등급. 절대 1위는 포화 가능(한계수익 체감). 시장 대비 우수라 증액 여력 검증된 채널을 선택.
